# 홈페이지 v2 컴포넌트 설계 및 다국어 키

## 1) 컴포넌트 계층
```text
AppLayout
├─ SiteHeader
│  ├─ NavMenu
│  ├─ LanguageSwitcher
│  └─ PrimaryCTA
├─ Main
│  ├─ HomePage
│  ├─ CareerPage
│  ├─ BlogListPage
│  ├─ BlogDetailPage
│  └─ ContactPage
└─ SiteFooter
```

## 2) 공통 컴포넌트

### `SiteHeader`
- 역할: 글로벌 탐색/언어 전환
- props:
  - `currentLang: "ko" | "ja" | "en"`
  - `currentPath: string`

### `LanguageSwitcher`
- 역할: 현재 페이지 경로를 유지한 채 언어 전환
- 규칙:
  - `/ko/career` -> `/en/career` 같은 매핑 유지
  - 없는 경로는 각 언어 홈으로 폴백

### `SectionTitle`
- 역할: 페이지 섹션 타이틀 공통 스타일
- props:
  - `titleKey: string`
  - `subtitleKey?: string`

### `TagBadge`
- 역할: 카테고리/역할/스킬 표시
- props:
  - `label: string`
  - `variant: "primary" | "neutral" | "outline"`

### `EmptyState`
- 역할: 필터 결과 없음 처리
- props:
  - `titleKey: string`
  - `descKey?: string`

## 3) 페이지 전용 컴포넌트

### Home
- `HeroCard`: 이름/요약/CTA
- `StrengthGrid`: 강점 카드 3개
- `FeaturedCareerList`: 대표 경력 프리뷰
- `LatestPostList`: 최신 글 프리뷰

### Career
- `CareerFilterBar`: 카테고리/직책/규모/기간 필터
- `CareerTimeline`: 타임라인 렌더링
- `CareerTimelineItem`: 단일 경력 카드
- `CareerHighlights`: 승진/리모델링 등 이벤트 요약

### Blog
- `BlogCategoryTabs`: 카테고리 탭
- `BlogSearchBar`: 검색/태그
- `BlogPostCard`: 목록 카드
- `BlogPagination`: 페이지 이동
- `BlogToc`(선택): 상세 글 목차

### Contact
- `ContactMethodCard`: 채널별 연락 정보
- `ContactNotice`: 문의 안내 문구

## 6) 언어 운영 정책
- 기본 언어: 한국어(`ko`)
- 언어 코드: `ko`, `ja`, `en`
- 번역 우선순위:
  1. 내비게이션/버튼/라벨
  2. 페이지 소개 문구
  3. 블로그/경력 데이터의 설명 텍스트
- 데이터 원문 규칙:
  - CSV 원문 텍스트는 원문 유지
  - UI 라벨/설명 문구만 번역

## 7) 접근성/품질 기준
- 키보드 탭 이동 가능
- 언어 전환 컨트롤 `aria-label` 제공
- 색상 대비 WCAG AA 최소 충족
- 텍스트 리소스 누락 시 `ko` 폴백
